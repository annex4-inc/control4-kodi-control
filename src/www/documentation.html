Kodi

Description

This driver enables control over Kodi. The driver provides various programming events, button bindings, commands, and a full mapping of control inputs. Communication to Kodi is over a Websocket and HTTP connection.

Features

  • Supports various programming events for screensaver, transport control, and playlist changes
  • Supports Websocket and HTTP communication
  • Supports button remapping for buttons that don't have a logical default control
  • Supports 5 useful device commands, 1 to display a notification and another to play a favorited movie by title

Driver Installation

  1. Place the driver into the project
  2. Set the IP address in the network tab
  3. Bind audio and/or video connections
  4. Refresh navigators

Kodi Installation

  1. Go to Settings
  2. Go to Services
  3. Set the access level (bottom left) to at least 'Standard'
  4. Select Control
  5. Enable 'Allow remote control via HTTP'
  6. Set port to '8080'
  7. Verify no password is set
  8. Enable 'Allow remote control from applications on this system'
  9. Enable 'Allow remote control from applications on other systems'

Workarounds

This driver manages a Websocket and HTTP connection to your Kodi device as there is a known issue in Kodi that causes a connection to block when a dialog window opens. When the issue occurs, the driver will detect the blocked connection and finish the dialog window interaction with the other connection. In terms of functionality you won't notice anything
